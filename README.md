==============================================================================

An [Ember.js](https://www.emberjs.com) shim for [A-Frame](https://aframe.io)

Installation
------------------------------------------------------------------------------

```
ember install ember-aframe-shim
```


Usage
------------------------------------------------------------------------------

```js
import AFRAME from 'aframe';
import THREE from 'three';

console.log(AFRAME.version);
console.log(THREE.REVISION);
```
