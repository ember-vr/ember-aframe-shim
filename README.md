An Ember.js shim for [A-Frame](https://aframe.io).

## Installation

`ember install ember-aframe-shim`

## Usage

```js
import Ember from 'ember';
import AFrame from 'aframe'

export default Ember.Component.extend({
  version: AFrame.version
});
```
